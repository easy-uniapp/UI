## Install

```sh
# use yarn
yarn add @easy-uniapp/ui

# use npm
npm install @easy-uniapp/ui --save
```

Add sass support.

```sh
# use yarn
yarn add sass sass-loader --dev

# use npm
npm install sass sass-loader --save-dev
```

## Usage

```scss
@import '~@easy-uniapp/ui';
```
